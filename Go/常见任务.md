# 常见任务

1. 内置json解析
利用反射实现，通过FeildTag来标识对应的json的值
### 
```go
type BasicInfo struct {
    Name string `json:"name"`
    Age int `json:"age"`
}
type JobInfo struct {
    Skills []string `json:"skills"`
}
type Employee struct {
    BasicInfo BasicInfo `json:"basic_info"`
    JobInfo JobInfo `json:"job_info"`
}
```
